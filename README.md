## ✨ Key Features
- **Glassmorphism Design**: Modern, semi-transparent UI elements for a high-end feel.
- **Dynamic Backgrounds**: Animated blobs and gradients that provide a premium aesthetic.
- **Scroll Animations**: Smooth entry animations for sections and cards to engage visitors.
- **Fully Responsive**: Optimized for every screen size—from mobile phones to 4K monitors.
- **Multi-Page Architecture**: Separate logic for Home, Projects, and Contact for a professional site structure.
- **Lucide Icon Integration**: Crisp, vector-based icons for all UI elements.

## 🛠️ Quick Customization Guide

1.  **Identity**: Search (Ctrl+F) for `[Your Name]` across the `.html` files and replace it with your own name.
2.  **Documents**: Ensure your PDF is named `Résumé.pdf` or update the links in the HTML.
3.  **Visuals**: Modify the `--primary` and `--secondary` colors in `style.css` to match your personal brand.
4.  **Socials**: Update the `mailto:` link and social media URLs in `contact.html`.

---

## 🌐 Deployment to GitHub Pages
1.  **Push** these files to your GitHub repository named `JCA`.
2.  Navigate to **Settings** > **Pages**.
3.  Under **Build and deployment**, select `Deploy from a branch`.
4.  Set **Branch** to `main` and **Folder** to `/ (root)`.
5.  **Save** and wait 60 seconds. Your portfolio is now live!
